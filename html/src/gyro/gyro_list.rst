.. _cgyro_amp:

AMP
---

**Definition**

Initial amplitude of finite-:math:`n` modes.

**Comments**

- DEFAULT = 0.1
- For linear simulations, the value is unimportant
- For nonlinear runs, this will usually need to be reduced to a smaller value.
